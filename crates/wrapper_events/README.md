# unified-agent-api-wrapper-events

Shared ingestion primitives for wrapper JSONL and NDJSON outputs used by the
`unified-agent-api` workspace.

- crates.io package: `unified-agent-api-wrapper-events`
- Rust library crate: `wrapper_events`

This crate provides bounded-memory line ingestion, optional async helpers, and
feature-gated adapters for the `codex` and `claude_code` wrapper libraries.
